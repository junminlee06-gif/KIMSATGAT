# 삿갓유람: 이매망량 포획첩

조선의 방랑시인 김삿갓이 전국을 유람하며 민담 속 환상동물을 마주하고, 퇴치·포획·진정하는 한지 액션 RPG 원형입니다.

## 현재 빌드

- 단일 HTML 웹 게임 원형
- GitHub Pages 실행 가능 구조
- 한지·종이 오려붙이기풍 그래픽 방향
- WASD/방향키 이동
- Space 대쉬 및 상호작용
- Q 기본 공격
- E 시조파동 스킬
- R 포획
- 실시간 전투 시스템
- 미니맵
- 포획첩 기록
- 여러 지역 이동 구조
- 일반 환수 리젠
- 보스 환수 무호, 강철이 원형

## 게임 방향

이 프로젝트는 추리물보다 액션 RPG 구조에 초점을 둔 김삿갓 유람 게임 원형입니다.

핵심 방향은 다음과 같습니다.

- 조선 민속과 환상동물 기반의 어두운 유랑 서사
- 한지, 먹선, 종이 질감 중심의 화면 구성
- 필드 이동과 실시간 전투가 이어지는 단일 웹 게임
- 환수를 단순 처치 대상이 아니라 포획하거나 진정시킬 수 있는 존재로 표현
- 장마다 새로운 마을, 사냥터, 보스 환수를 추가하는 확장형 구조

## 주요 조작

- 이동: WASD / 방향키
- 대쉬 및 상호작용: Space
- 상호작용: Enter / Z
- 기본 공격: Q
- 시조파동: E
- 포획: R
- 회피: Space

## 실행 방법

브라우저에서 `index.html`을 열면 실행됩니다.

GitHub Pages를 사용할 경우, 저장소 설정에서 Pages 배포 소스를 `main` 브랜치의 root로 지정하면 됩니다.

배포 주소 예시:

```txt
https://junminlee06-gif.github.io/KIMSATGAT/
```

## 작업 메모

이 저장소는 김삿갓 유람 액션 RPG의 세계관·전투·맵 구조를 계속 다듬기 위한 프로토타입 저장소입니다.

현재 우선순위는 다음과 같습니다.

1. 한지 그래픽의 밀도와 캐릭터 가독성 개선
2. 실시간 전투 타격감과 적 패턴 보강
3. 환수별 고유 이펙트와 포획 연출 추가
4. 장 단위 진행 구조 정리
5. GitHub Pages 배포 안정화
